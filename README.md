RakutenCall
===========

###【アプリ概要】###
このアプリは楽天でんわを使っている人向けのアプリです。  
このアプリでは以下のことができます。

+ 発信時に自動で「0037-68」をつけてそのまま電話をかけてくれる
+ 上記の変換対象に含めない電話番号を登録できる
+ 設定からデフォルトで起動する電話アプリを設定できる

###【現在報告されている不具合】###
